# Wedding predicrtion

## Overview
This project predicts whether a text description contains a marriage-related mention (`txtMarr`) using:
- Bag-of-Words features
- Logistic Regression / L1-regularized Logistic Regression
- Neural Networks

Main notebook: `Benyazza02.ipynb`


## Dataset
Notebook expects:
- `Data/Letterboxd_annotations - Letterboxd_annotations.csv`

## How to run
1. Open `Benyazza02.ipynb`
2. Run cells from top to bottom
3. Check model comparison table in the conclusion section
